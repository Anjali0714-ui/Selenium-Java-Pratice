# Selenium Java Practice Scripts

This repository contains Selenium WebDriver automation scripts written in Java to practice UI automation concepts.

## Covered Topics
- Alerts handling
- Dropdowns
- Radio buttons
- Explicit and Implicit waits
- Mouse actions
- Window handling
- Popup handling
- TestNG execution

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Eclipse
